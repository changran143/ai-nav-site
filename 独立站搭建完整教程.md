## 📍 第1步：购买域名和服务器

### 1.1 域名购买（阿里云/腾讯云/其他）

**推荐平台**：阿里云（wanwang.aliyun.com）

**操作步骤**：
1. 访问阿里云域名注册页面
2. 搜索你想要的域名，如 `penguinai.com` / `aibase.cn` / `ai-tools.cn`
3. 建议选择：
   - `.com` 或 `.cn` 后缀
   - 简短好记
   - 与品牌相关
4. 加入购物车，完成实名认证后付款
5. 价格参考：
   - `.com`：约 ¥60-80/年
   - `.cn`：约 ¥30-40/年

**建议域名示例**：
- penguinai.com
- penguinai.cn
- ai-base.cn
- ai-tools-nav.com

---

### 1.2 服务器购买（推荐配置）

**推荐**：阿里云 ECS 或 腾讯云 CVM

**入门配置**（足够支撑初期1-2万用户）：
```
CPU: 2核
内存: 4GB
硬盘: 40GB SSD
带宽: 3-5Mbps
系统: Ubuntu 22.04 LTS
```

**价格参考**：
- 阿里云：约 ¥200-300/年（新用户特惠）
- 腾讯云：约 ¥150-250/年（新用户特惠）

**购买步骤**：
1. 登录阿里云/腾讯云控制台
2. 搜索「云服务器 ECS」
3. 选择地区（建议选离你最近的，如华东/华南）
4. 选择配置：2核4G，Ubuntu 22.04
5. 购买时长：1年
6. 付款并记住你的服务器 IP 地址

---

### 1.3 域名备案（国内服务器必须）

⚠️ **重要**：使用国内服务器必须备案！

**备案流程**：
1. 购买域名后，在阿里云/腾讯云备案系统提交申请
2. 填写网站信息（个人备案需要身份证照片）
3. 等待初审（1-3个工作日）
4. 初审通过后，进行工信部短信核验
5. 管局审核（约7-20个工作日）
6. 备案成功，获得备案号

**加速技巧**：
- 如果不想备案，可购买香港/新加坡服务器（免备案但速度稍慢）
- 先使用香港服务器上线，同时准备国内备案

**安装命令**（Ubuntu系统）：

## 📍 第5步：配置网站和SSL

### 5.1 添加网站

**宝塔面板**：
1. 左侧 → 网站 → 添加站点
2. 域名：输入你购买的域名
3. 根目录：`/www/wwwroot/ai-nav-site`
4. PHP版本：纯静态
5. 点击「提交」

### 5.2 配置反向代理

让 Nginx 把请求转发给 Flask：

1. 点击网站名 → 设置
2. 配置文件，添加：

```nginx
location / {
    proxy_pass http://127.0.0.1:5000;
    proxy_set_header Host $host;
    proxy_set_header X-Real-IP $remote_addr;
}

location /static {
    alias /www/wwwroot/ai-nav-site/static;
}
```

### 5.3 申请SSL证书（HTTPS）

1. 网站设置 → SSL
2. 选择「Let's Encrypt」
3. 勾选域名 → 点击「申请」
4. 开启「强制HTTPS」
