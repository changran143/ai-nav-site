## 📍 第2步：配置域名解析（购买后立即做）

### 2.1 添加DNS解析

**操作步骤**：
1. 阿里云控制台 → 域名 → 找到你的域名 → 解析
2. 点击「添加记录」：
   - 记录类型：A
   - 主机记录：@（表示主域名）
   - 解析线路：默认
   - 记录值：你的服务器IP（在轻量服务器控制台查看）
   - TTL：10分钟
3. 再添加一条：
   - 记录类型：A
   - 主机记录：www
   - 记录值：同上

**验证**：
- 打开 CMD，输入：`ping 你的域名`
- 能看到返回的服务器IP就对了（DNS生效需要10分钟-2小时）

## 📍 第10步：配置网站和SSL（第2-3天）

### 10.1 添加网站

**宝塔操作**：
1. 宝塔 → 网站 → 添加站点
2. 填写：
   - 域名：你的域名（如 penguinai.cn）
   - 根目录：`/www/wwwroot/ai-nav-site`
   - PHP版本：纯静态
3. 点击「提交」

### 10.2 配置反向代理

1. 点击网站名 → 设置 → 反向代理
2. 添加反向代理：
   - 目标URL：`http://127.0.0.1:5000`
   - 发送域名：`$host`
3. 点击「保存」

### 10.3 配置静态文件

1. 网站设置 → 配置文件
2. 在 `server` 块中添加：

```nginx
location /static {
    alias /www/wwwroot/ai-nav-site/static;
    expires 30d;
}

location /uploads {
    alias /www/wwwroot/ai-nav-site/uploads;
    expires 30d;
}
```

3. 点击「保存」

### 10.4 申请SSL证书

1. 网站设置 → SSL
2. 选择「Let's Encrypt」
3. 勾选你的域名
4. 点击「申请」
5. 申请成功后，开启「强制HTTPS」

## 🆘 常见问题速查

### Q: 宝塔登录不了？
A: 检查安全组是否开放8888端口。轻量服务器控制台 → 防火墙 → 添加规则：端口8888

### Q: 域名解析不生效？
A: 等待10分钟-2小时，或用 `ping 域名` 测试

### Q: 网站显示502错误？
A: Python项目没启动，或端口不对。检查「Python项目管理器」日志

### Q: 静态文件加载不了？
A: 检查nginx配置中的 `/static` 路径是否正确

### Q: 数据库连接失败？
A: 检查数据库密码、用户名是否正确，PostgreSQL是否已启动
